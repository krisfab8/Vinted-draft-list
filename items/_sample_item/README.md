# Sample Item Folder

This is an example of how to structure a new item folder.

## Folder Naming Convention
Use a short, descriptive name with no spaces:
```
items/barbour-wax-jacket-olive-c42/
items/brora-lambswool-navy-12/
items/flannel-shirt-check-m/
```

## Required Photos
| Filename      | What to capture                          |
|---------------|------------------------------------------|
| front.jpg     | Full front of garment, flat or hanging   |
| tag.jpg       | Brand/size label — close up, well lit    |
| material.jpg  | Material composition label               |
| back.jpg      | Full back of garment                     |

## Optional Photos (upload to Vinted but not analyzed)
| Filename      | What to capture                          |
|---------------|------------------------------------------|
| detail_1.jpg  | Interesting feature (pocket, collar etc) |
| detail_2.jpg  | Any visible flaws (if present)           |
| measurements.jpg | Flat lay with tape measure (optional) |

## Notes
- Shoot in good natural light
- White/neutral background preferred
- HEIC files from iPhone are fine — system will handle them
- You can name extra photos anything — all files in the folder get uploaded
- Only front.jpg, tag.jpg, material.jpg, back.jpg are sent to the AI for analysis
