# Title Format
Brand + Item Type + Key Feature + Colour + Size

# Description Format
Short opening sentence (what it is, brand, colour).
Key features in short lines (material, notable details, origin if interesting).
Condition sentence — honest, no hype.
Measurements in photos.
Fast postage.

# Tone
Concise, clean, non-hype.

# Example (Men's)
Barbour wax jacket in olive green with corduroy collar.

- Men's size C42
- Waxed cotton outer
- Made in England

Good used condition with normal signs of wear.
Measurements in photos.
Fast postage.

# Example (Women's)
Brora lambswool rollneck in oatmeal.

- Women's size 12
- 100% lambswool
- Ribbed collar and cuffs

Excellent condition, barely worn.
Measurements in photos.
Fast postage.

# Rules
- Title max 80 characters.
- Do not use ALL CAPS.
- Do not use "rare", "stunning", "amazing", "perfect" unless objectively accurate.
- If there are flaws, state them plainly in the description.
