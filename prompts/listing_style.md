# Title Format
Brand + Item Type + 1–2 Search Keywords + Colour + "size" + UK Size

## Keyword Strategy — use terms buyers actually search
Pick the 1–2 most relevant from these categories (accurate, visible only):

**Fabric/material:** wool, cashmere, waxed cotton, tweed, corduroy, merino, lambswool
**Style/fit:** slim fit, regular fit, single breasted, double breasted, button up, rollneck
**Use case:** smart casual, business, office, formal — only if item is clearly suited to it
**Origin/provenance:** Italian, Made in England, Made in Italy — only if visible on tag
**Notable feature:** corduroy collar, patch pockets, birdseye weave, pinstripe

- Always write "size" before the size: e.g. "size 44R", "size W32 L32", "size M"
- For trousers/jeans/shorts: ALWAYS show waist + length as "W32 L32". If only waist is on tag, write "W32". Never use just a bare EU number for trousers.
- For suits/blazers with bare EU number (e.g. "54"): convert to UK by subtracting 10, add R. e.g. EU 54 → UK 44R
- For suits/blazers already in UK format with suffix (e.g. "44R"): keep as-is — it is already the UK chest size
- For other items: use size as printed on tag (e.g. C42, 12, M, L)

e.g. "Ermenegildo Zegna wool cashmere slim blazer charcoal size 44R"
e.g. "Suit Supply Brentwood slim wool trousers grey W32 L32"
e.g. "Barbour wax jacket corduroy collar olive size C42"
e.g. "Hugo Boss slim fit business blazer navy size 42R"
e.g. "John Smedley merino rollneck smart casual navy size M"

# Description Format
Short opening sentence (what it is, brand, colour).
Key features in short lines (material, notable details, origin if interesting).
Condition line — format: "[Condition] — [brief note e.g. no stains or holes / light wear to X]".
Measurements in photos.
Fast postage.

# Tone
Concise, clean, non-hype.

# Example (Men's)
Barbour wax jacket in olive green with corduroy collar.

- Men's size C42
- Waxed cotton outer
- Made in England

Very good used condition — no stains or holes.
Measurements in photos.
Fast postage.

Very good condition — barely worn.
Measurements in photos.
Fast postage.

# Rules
- Title max 80 characters.
- Do not use ALL CAPS.
- Do not use "rare", "stunning", "amazing", "perfect" unless objectively accurate.
- Default condition is "Very good" unless photos show clear heavy wear or damage.
- Condition line MUST start with "Very good" — never use "Good vintage", "Good used", or similar. Wrong: "Good vintage condition — minimal wear." Correct: "Very good used condition — no stains or holes."
- If there are flaws, state them plainly in the description.
